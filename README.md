# Plataforma de Captação e Estruturação de Projetos Tecnológicos

Projeto desenvolvido no programa Bolsa Futuro Digital.

## Objetivo

Construir uma plataforma capaz de:

* captar demandas reais de empresas;
* estruturar projetos tecnológicos;
* alimentar futuras turmas do programa.

## Estrutura

```text
/frontend → Aplicação frontend
/backend  → API e backend
/docs     → Documentação
/assets   → Arquivos visuais
/infra    → Infraestrutura e deploy
```

## Tecnologias

### Frontend

* React
* Next.js

### Backend

* FastAPI
* Python

### Banco de dados

* Supabase
* PostgreSQL

### Infraestrutura

* GitHub
* Vercel
* Render
